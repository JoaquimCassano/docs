## Instalando
Antes de instalarmos, é necessário que tenhamos instalado:
- [Python](https://www.python.org/downloads/)

Para instalar a Alertpix.py, basta digitar o seguinte comando em seu terminal:
```
python3 -m pip install alertpix
```

Ou instale pelo código-fonte:
```
git clone https://github.com/JoaquimCassano/Alertpix.py
cd Alertpix.py
python3 -m pip install .
```
