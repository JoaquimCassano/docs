Nesse projeto, criaremos um código que mostrará o saldo de sua conta na Alertpix

Para iniciar, crie um arquivo python chamado `exemplo-saldo.py` em seu editor de código favorito e siga os passos a seguir:

<Steps>
  <Step title="Importar a SDK">
    Para isso, insira a seguinte linha no seu arquivo:
    ```python
    import alertpix
    ```
    <Warning>
      Para funcionar, é necessário que a Alertpix.py esteja instalada em seu computador.
      Não instalou? Siga o tutorial na [introdução](/introduction.mdx)
    </Warning>
  </Step>
  <Step title="Configurar o client">
      Agora que já importamos a SDK, vamos "arrumar a casa", isto é, criarmos a [classe](https://docs.python.org/3/tutorial/classes.html) do Client
      ```python
      token:str = "Bearer <seu token aqui>"
      client = alertpix.Alertpix(token)
      ```
      <Info>
        Substitua `"Bearer <seu token aqui>"` por sua própria api key. Não tem uma ainda? Consiga em https://TODO_CHANGELATER.com
      </Info>

  </Step>
  <Step title="Fazer a requisição e mostrar o resultado">
    Agora, vamos colocar a mão na massa.
    Insira o seguinte código no arquivo:
    ```python
      saldo, saldoBruto = client.account.wallet
      print(f'Saldo: {saldo}')
      print(f'Saldo bruto: {saldoBruto}')
    ```

  </Step>
</Steps>

Portanto, o seu código deve estar assim:

```python
import alertpix
token:str = "Bearer <seu token aqui>"
client = alertpix.Alertpix(token)

saldo, saldoBruto = client.account.wallet
print(f'Saldo: {saldo}')
print(f'Saldo bruto: {saldoBruto}')
```
Para testarmos, execute o código rodando o seguinte comando:
```
python3 exemplo-saldo.py
```
